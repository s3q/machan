# machan
A tool used in spoofing mac address.. The tool is based on Python


# How do you use :

python3 index.py --help

-i    ->    interface

-m    ->    new mac

-r    ->    select a file for do random mac

python3 index.py -i eth0 -m 00:00:00:00:00:00

python3 index.py -i eth0 -r macaddr.txt
